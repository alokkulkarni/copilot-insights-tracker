# Mobile Checklist

## Breakpoints

- [ ] 640px — single-column stats grid, compact padding
- [ ] 768px — hamburger nav, stacked table header toolbar
- [ ] 1024px — single-column charts grid
- [ ] 1280px — two-column charts grid

## Touch Targets

- [ ] All buttons ≥ 44×44px (WCAG 2.5.5)
- [ ] All nav links ≥ 44px height
- [ ] Form inputs ≥ 44px height

## Layout

- [ ] No horizontal scroll on any viewport
- [ ] Tables scroll horizontally within their container (not the page)
- [ ] Charts resize correctly on small screens

## Performance

- [ ] First Contentful Paint < 2s on 3G
- [ ] No layout shift (CLS < 0.1)
